# Contributing to GIF

Contributions are welcome — bug reports, fixes, documentation improvements, and tests.

## Reporting issues

Open a [GitHub Issue](https://github.com/jgamboa-biotecno/GIF-Framework/issues) including:

- A description of the problem and what you expected to happen
- Steps to reproduce
- The GIF version (`gif --version`)
- Any relevant assembly accession(s)

## Pull requests

1. Fork the repository and create a feature branch
2. Make your changes and add tests where applicable
3. Run the test suite locally: `pytest`
4. Open a pull request describing the change and its motivation

## Scoring algorithm changes

Modifications to the V/P/C/R scoring components affect the reproducibility of results reported in publications. If you are proposing such a change, please open an issue first describing the rationale and the expected impact on existing results.
